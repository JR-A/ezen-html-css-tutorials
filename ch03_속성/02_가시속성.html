가시 속성

태그가 화면에 보여지는 방식을 지정

display 속성

block : 공간 개념으로 행단위 영역을 차지하며 폭, 높이 설정 가능
inline-block : 이미지와같이 폭, 높이는 가지면서 자리는 글자처럼 차지
inline : text 입력한만큼만 자리 차지. 폭,높이 없음
none : 화면상에 보이지 않음. 차지하고 있던 공간 자체도 사라짐

visibility 속성

visible 화면에 보임
hidden 화면에서 숨겨짐. 원래 있던 공간은 그대로 차지

opacity 속성

투명도. 0(투명)~1(불투명) 사이의 소수로 표현
