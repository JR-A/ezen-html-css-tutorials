폰트 속성

종류

font-family : 서체, 서체, ("한글 서체"), 한글서체 영문명, 기본서체제공
font-size : 글자크기
font-weight : 두께 normal(보통) bold(두껍게) - (**strong, b, h1~h6 기본으로 두꺼운 서체 초기화시 normal**)
font-style : 기울기 normal(기울기없음) italic(기울어짐) (**em, i, address 기본으로 기울어진 서체 초기화시 normal**)
color : 글자색
line-height : 줄간격 normal %
letter-spacing : 자간
word-spacing : 단어 간격
text-align : 글자의 정렬 left(왼쪽정렬) center(가운데정렬) right(오른쪽정렬) justify(양쪽정렬)
text-decoration : 밑줄 none(밑줄없음) underline(밑줄) (**a 기본으로 밑줄쳐진 서체 초기화시 none**)

font속성에 한번에 값 입력할 경우

font: style weight size / lined-height family(서체1), family(서체2), family(서체3); color: 글자색;
