CSS 단위

크기 단위

px(픽셀), %, rem, em(배수단위), vw(전체 노출되는 화면기준 너비에 맞춤), vh(전체 노출되는 화면기준 높이에 맞춤), pt, ex, cm, mm, in(인치)....
주로 많이 사용 : px % rem em vw vh

색상 단위

색상명, 16진수 색상코드(HEX단위), RGB단위, RGBA단위, HSL단위, HSLA단위 등...
주로 많이 사용 : HEX, RGBA

색상명

-> red, orange, pink...

HEX단위

-> #RRGGBB -> #ff0000(#f00)

RGB단위

-> rgb(red, green, blue) -> rgb(255,0.0);

RGBA단위

-> rgba(red, green, blue, alpha) -> rgba(255,0,0,0.5); ** alpha : 투명도 **

HSL단위

-> hsl(hue, saturation, lightness) -> hsl(0,100%,50%);

HSLA단위

-> hsla(hue, saturation, lightness, alplha) -> hsl(0,100%,50%, 0.5);

***HEX단위 표기법***
#RRGGBB = #RGB
Red Green Blue - 빛의 3원색
각 자리별로 색상 표현 - 0~9 A~F
-검정 #000000 -> #000 반복되는 것 축약가능
-흰색 #ffffff -> #fff
-빨강 #ff0000 -> #f00
-#ff9a00 -> 세자리로 쓸 수 없음

URL 단위

절대경로, 상대경로

--상대경로 : 현재 문서 기준으로 보여주고자 하는 이미지, 링크된 html문서의 상대적 경로
현재폴더/file.확장자 - 현재 문서 기준 같은 폴더에 위치한 file
./file.확장자 - 현재 문서 기준 같은 폴더에 위치한 file
../file.확장자 - 현재 문서 기준 상위 폴더에 위치한 file

--절대경로 : 보여주고싶은 파일이나 문서의 컴퓨터상 경로.
