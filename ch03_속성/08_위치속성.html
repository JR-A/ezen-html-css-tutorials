위치 속성

position, float

position 속성

static : 기본 값. 위에서 아래 순서대로 배치
relative : absolute의 기준 left right top bottom 같이 사용 가능하지만 웬만하면 쓰지 말 것.
absolute : relative(기준)이 없다면 웹브라우저 기준으로 left right top bottom 적용가능
fixed : 본문의 길이, 스크롤의 위치와 상관없이 위치 고정된 left right top bottom 적용가능

left right top bottom 속성

단독 사용 불가
반드시 position 속성 적용된 부분에만 추가로 작성
방향별로 크기 단위 입력시 위치 변경
margin-left margin-right margin-top margin-bottom과는 개념 다름 -> 떨어진 거리 개념

z-index

단독 사용 불가 반드시 position 속성 적용된 부분에만 추가로 작성
콘텐츠 입력 순서상 마지막에 입력된 영역이 가장 우선순위 높다 -> z-index가 이 우선순위 바꾸는 역할
입력 순서 상관 없이 우선순위 변경함

position 속성 작성 규칙

1. absolute 의 기준이 되는 relative는 height속성을 반드시 포함해야함.
2. absolute 의 기준은 직계 부모에만 작성할 수 있음.

float 속성

left : 왼쪽 고정
right : 오른쪽 고정

이전에 작성된 형제 영역에 적용된 float에 영향 받지 않으려면

clear속성에 left right both 중 하나 입력

overlow 속성

scroll : 기본값. 영역 밖을 벗어난 부분을 스크롤로 보여줌
hidden : 영역 밖을 벗어났을 때 숨김처리 / 자식 float이 적용된 경우 부모 영역 확보

float 작성 규칙

1. 자식에 float이 적용될 경우, 부모에 overflow: hidden 또는 height적용 또는 float 적용 2. 이전 형제에 float 적용될 경우, 나머지 공간에 다음에 오는 영역이 채워진다. ->영향 받지 않기 위해 clear 속성 입력
