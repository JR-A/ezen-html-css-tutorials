박스속성

종류

width : 폭(너비)
height : 높이
padding : 안쪽 여백
border : 테두리
margin : 주변 여백

width, height 속성

입력 : 크기 단위 (px % em vw vh)

padding 속성

padding-top
padding-left
padding-right
padding-bottom
padding

ex)
padding: 10px; -> 상 10px 우 10px 하 10px 좌 10px (같은값이 사방으로 적용)
padding: 0 10px; -> 상 0px 우 10px 하 0px 좌 10px (상하0px, 좌우10px)
padding: 0 10px 20px; -> 상 0px 우 10px 하 20px 좌 10px (상0px, 좌우10px, 하20px)
padding: 0 10px 20px 30px; -> 상 0px 우 10px 하 20px 좌 30px (시계방향)

border 속성 : 3 속성이 모두 입력되어야 정확히 출력됨

border-width : 두께(크기 단위 입력)
border-color : 색상(색상 단위 입력)
border-style : 테두리 스타일

주로 한 번에 입력
ex) border: width style color; (2px 실선 빨강: 2px solid #f00;)

방향별 border 설정시 border-top, border-right, border-bottom, border-left 시계방향

border-style

none : 없음
solid : 실선
dashed : 대시선
dotted : 점선
double : 이중선
outset : 양각선
inset : 음각선
ridge : 테두리만 양각선
groove : 테두리만 음각선

border-radius

박스모델 모서리 부분을 둥글게 만들 때.

border-radius : 좌상 우상 우하 좌하; (왼쪽 상단 기준 시계방향)
동일 값 입력할때는 값 하나만 입력하면 됨.
